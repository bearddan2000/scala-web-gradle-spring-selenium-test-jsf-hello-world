# scala-web-gradle-spring-selenium-test-jsf-hello-world

## Description
A springboot secure web app with jsf support.

## Tech stack
- scala
- sbt
- scala
- gradle
  - springboot
  - jsf

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
